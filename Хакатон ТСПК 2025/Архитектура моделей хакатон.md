# City

Модель для хранения информации о городах, в которых проводятся посадки.

```python
class City(models.Model):
    name = models.CharField(max_length=255, verbose_name="Название города")
    is_active = models.BooleanField(default=True, verbose_name="Активен")

    def __str__(self):
        return self.name

    class Meta:
        verbose_name = "Город"
        verbose_name_plural = "Города"
```

- `name` - Название города.
- `is_active` - Флаг, указывающий, активен ли город (отображается ли на сайте).

# ParticipationFormat

Модель для хранения информации о форматах участия в посадках (например, "Частное лицо", "Компания", "Ритуалы").

```python
class ParticipationFormat(models.Model):
    name = models.CharField(max_length=255, verbose_name="Название формата")
    slug = models.SlugField(unique=True, verbose_name="URL", help_text="Автоматически генерируется из названия")
    description = models.TextField(blank=True, null=True, verbose_name="Описание")
    image = models.ImageField(upload_to="participation_formats/", blank=True, null=True, verbose_name="Изображение")
    is_company = models.BooleanField(default=False, verbose_name="Для компаний")
    is_ritual = models.BooleanField(default=False, verbose_name="Ритуал")
    order = models.IntegerField(default=0, verbose_name="Порядок отображения")

    def __str__(self):
        return self.name

    def save(self, *args, **kwargs):
        if not self.slug:
            self.slug = slugify(self.name)
        super().save(*args, **kwargs)

    class Meta:
        verbose_name = "Формат участия"
        verbose_name_plural = "Форматы участия"
        ordering = ["order"]
```

- `name` - Название формата участия.
- `slug` - версия названия для использования в URL.
- `description` - Описание формата участия.
- `image` - Изображение для формата участия.
- `is_company` - Флаг, указывающий, предназначен ли этот формат для компаний.
- `is_ritual` - Флаг, указывающий, является ли этот формат "ритуальным".
- `order` - Порядок отображения форматов.

# PrivatePersonApplication

Модель для хранения заявок от частных лиц.

```python
class PrivatePersonApplication(models.Model):
    name = models.CharField(max_length=255, verbose_name="Имя")
    phone = models.CharField(max_length=20, verbose_name="Телефон")
    email = models.EmailField(verbose_name="Email")
    city = models.ForeignKey(City, on_delete=models.CASCADE, verbose_name="Город")
    participation_format = models.ForeignKey(ParticipationFormat, on_delete=models.CASCADE, verbose_name="Формат участия")
    saplings_quantity = models.IntegerField(verbose_name="Количество саженцев")
    application_date = models.DateTimeField(auto_now_add=True, verbose_name="Дата заявки")

    def __str__(self):
        return f"Заявка от {self.name} ({self.email})"

    class Meta:
        verbose_name = "Заявка от частного лица"
        verbose_name_plural = "Заявки от частных лиц"
        ordering = ["-application_date"]
```

- `name` - Имя.
- `phone` - Телефон.
- `email` - Email.
- `city` - Город (ссылка на модель `City`).
- `participation_format` - Выбранный формат участия (ссылка на модель `ParticipationFormat`).
- `saplings_quantity` - Количество саженцев.
- `application_date` - Дата подачи заявки.

# HomePageSection

Модель для хранения секций главной страницы.

```python
class HomePageSection(models.Model):
    SECTION_TYPES = [
        ('text', 'Текст'),
        ('video', 'Видео'),
        ('image', 'Изображение'),
    ]

    title = models.CharField(max_length=255, verbose_name="Заголовок")
    content = models.TextField(blank=True, null=True, verbose_name="Содержание")
    section_type = models.CharField(max_length=20, choices=SECTION_TYPES, default='text', verbose_name="Тип секции")
    video_url = models.URLField(blank=True, null=True, verbose_name="URL видео")
    image = models.ImageField(upload_to="home_page_sections/", blank=True, null=True, verbose_name="Изображение")
    order = models.IntegerField(default=0, verbose_name="Порядок отображения")
    is_active = models.BooleanField(default=True, verbose_name="Активна")

    def __str__(self):
        return self.title

    class Meta:
        verbose_name = "Секция главной страницы"
        verbose_name_plural = "Секции главной страницы"
        ordering = ["order"]
```

- `title` - Заголовок секции.
- `content` - HTML контент секции.
- `section_type` - Тип секции (текст, видео, изображение).
- `video_url` - URL для видео.
- `image` - Изображение для секции.
- `order` - Порядок отображения секций.
- `is_active` - Показывать ли секцию.
